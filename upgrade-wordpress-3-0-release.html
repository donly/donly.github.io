    升级 WordPress 3.0 正式版：备份文件和数据库

    WordPress是一个免费的开源项目,在GNU通用公共许可证下授权发布。目前最新版本为2010年6月18日发布的3.0版。

    经 RC3 测试版过后，WordPress 马上放出 3.0 正式版。登陆管理后台就可以看到更新提示了。作为一位版本控，二话不说马上准备升级。下面就简单介绍一下升级的准备工作。

    备份文件到本地机器上。可以使用 FTP 工具（FileZilla），或 UNIX Shell （比如，SSH）。由于我的 SSH 被禁用，所以使用比较通用的 FTP 工具 FileZilla 来备份整个 WordPress 站点：

    1. WordPress核心安装文件
    2. WordPress插件
    3. WordPress主题
    4. 图片和文件
    5. Javascripts, PHP脚本, 和其他代码文件
    6. 其他文件和静态页面

    现在的大部分主机供应商都备份整个服务器，所以也包括了我们的网站，比如DreamHost 的就可以在 CPanel>Dreamhost>Backup Your Account 找到。BTW，想成为一名优秀的站长，应该懂得如何备份和养成良好的经常备份的习惯。

    备份数据库。第一种方法，使用命令行的方式访问（前提条件是必须先安装好 MySQL 软件）：
    mysql -u username -ppassword -h hostname databasename
    （注意: 在 -p 和 password 之前是没有空格的！，另外说明一下，有些主机限制这种连接方式，请用下面这种的方法吧）

    第二种方法，使用最流行的数据库管理系统之一的 phpMyAdmin。一般主机提供商有提供该工具，在主机 Cpanel 管理后台可找到。万一没有找到，请 Google 吧。下面是操作步骤：

    1. 只勾选包含WordPress站点的数据表
    2. 勾选"Add DROP TABLE"
    3. 勾选"Complete inserts"
    4. 勾选"Save as File"
    5. 压缩那项勾选"None"，如果你的数据库很大，那么选择一种压缩方式。
    6. 点Go，然后数据就会保存到你的电脑中。

    把这些文件保存到一个安全的地方 !

    最后，回到后台放心地点自动升级，很快就可以看到升级到 3.0 后的效果——后台界面变成白灰色。需要注意的是，现在的 WordPress 3.0 正式版是国际版本，有限支持简单中文，本地化版本暂时没有发布。
