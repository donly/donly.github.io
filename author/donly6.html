    iWebKit使用体会

    iWebKit是一个帮助我们创建iPhone, iPod Touch和iPad的webapp。iWebKit很容易上手，对HTML的知识几乎没什么要求，只要略懂HTML的人不用多少分钟就可以写出很专业的用词界面的网站页面。容易使用，加载速度快和兼容性、扩展性好是最大的特点。

    One coin has two sides，iWebKit不好方面：

    • 没有详细的使用文档API，可能是因为它太简单了吧。
    • 想自定义样式或主题比较麻烦，使用的都是iOS系统的标准用词界面，不利于个性化。

    推荐iPhone Web Apps入门开发者使用。开发调试时可使用WebKit内核的现代浏览器，比如Safari、Chrome。不支持Firefox。相关的Web Apps框架还有iUI，JQTouch等。

    如何防止Mac OS复制文件时自动生成的.DS_Store

    .DS_Store (Desktop Services Store)，是Apple公司的操作系统创建的一个隐藏文件，里面保存着自定义目录的图标和背景图片等元信息。没错，听起来就好像Windows上的desktop.ini。

    当我们从Mac OS中复制文件到移动磁盘，然后从Windows操作系统读取的时候，可发现这些自动生成的文件。幸运地，我们可以利用命令配置或软件来防止这些.DS_Store文件生成：

    • 打开终端Terminal： defaults write com.apple.desktopservices DSDontWriteNetworkStores true
    • 或者使用blueharvest这款软件。

    万能的Mac OS截图

    Mac OS X 的截图很方便，下面是常用的截图快捷键。

    **cmd为Command键*

    • 截取全屏保存到桌面：cmd+shift+3
    • 截取区域保存到桌面：cmd+shift+4
    • 截取全屏保存到剪贴板：cmd+ctrl+shift+3
    • 截取区域保存到剪贴板：cmd+ctrl+shift+4
    • 带阴影效果的区域截图：cmd+shift+4+space

    如果截到的图为黑色或无内容，请确认是否已安装显卡驱动。如果驱动无解，可借助像EverNote带有截图功能的软件来实现，虽然EverNote是一款专业的记事软件。

    另外有一款专门用于iPhone、iTunes截图的工具——iPhone-Simulator Cropper v2.4，在iPhone真机上也可按Home+Power键进行截屏。
