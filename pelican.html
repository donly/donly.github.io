    Pelican


    Pelican is a static site generator, written in Python.

    目前最新版本为 3.2 (2013-04-24)

    写作流程

    1. 开启 virtualenv
    2. 进入工作目录
    3. $ make devserver
    4. ./develop_server.sh stop
    5. 发布直接使用配置好的 git comit & push.

    更多详细流程见 Getting started — Pelican 3 documentation
