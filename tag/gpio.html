    树莓派 GPIO 入门

    GPIO，通用型之输入输出（General Purpose I/O）的简称。树莓派内核中已经编译自带了 GPIO 的驱动，可以直接使用第三方函数库来进行编程。

    比较常用的第三方函数库有：

    1. Python GPIO
    2. wiringPi
    3. BCM2835 C Library
    4. Shell
