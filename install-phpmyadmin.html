    安装 phpMyAdmin

    phpMyAdmin 是管理 MySQL 最方便的 Web 应用，如果经常使用 MySQL ，phpMyAdmin 是必备工具。

    本文主要参考资料：Installation — phpMyAdmin 4.7.0-dev documentation。

    在 OS X 安装

    • 去官网下载 最新的版本
    • 解压到 ~/Sites/phpmyadmin
    • 在 phpMyAdmin 目录下创建一个可写的 config 目录
    • 打开 http://localhost/~[username]/phpMyAdmin/setup，加载配置，修改并保存
    • 将 config 下生成的 config.inc.php 移到 phpmyadmin 根目录下
    • 删除 config 目录

    例子：

    $ cd /Users/[username]/Sites/phpMyAdmin
$ mkdir config
$ chmod o+rw config
$ cp config.sample.inc.php config.inc.php
$ vi config.inc.php # 按官网推荐的配置
$ mv config.inc.php config
$ chmod o+w config/config.inc.php
$ mv config/config.inc.php .
$ chmod o-rw config.inc.php
$ rm -rf config

    2002 错误， 找不到 mysql.sock 的解决方法：

    $ sudo mkdir /var/mysql
Password:
$ sudo ln -s /tmp/mysql.sock /var/mysql/mysql.sock
