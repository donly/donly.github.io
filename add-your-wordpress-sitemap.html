    WordPress：为您的博客增加“网站地图”页面

    网站地图（Sitemaps）又称站点地图，是一个显示网站所有页面的页面。它是一个很友好的页面，向来访的朋友展示您的站点结构。还有一种站点地图，XML站点地图，专门给搜索引擎看的，SEO 优化必备文件。有很多创建 XML 站点地图的在线工具，比如 Google 的 WebMaster Tools。这篇文章主要介绍的是前者，使用 WordPress 的模板系统创建一个“网站地图”页面。

    第一步，创建自己的页面模板

    最简单的方法就是使用 FTP 或 SSH 工具，到当前使用的主题目录（/wp-content/themes/您的主题名/）下复制 paga.php 文件里面的内容到新建的 Sitemap.php 文件，并在头部增加以下代码：


    < ?php /* Template Name: Sitemap */ ?>

    以上代码的表示定义一个“Sitemap”模板，这样此模板就可以在后台“外观”—>“编辑”的模板列表中可见了，名字为文件的名字，除了 保留的文件名（不可用）以外。这样就增加了一个自定义的模板了。
