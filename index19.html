    老生常谈：CSS overflow 那些事儿

    css 的 overflow 属性的用法很广泛也很重要，最近在修改 WordPress 和 CMS 的模板的时候就发现有很多地方是因为不正确地使用 overflow 属性而导致跨浏览器使用会出现诡异的渲染结果。

    overflow 属性定义和说明

    overflow 中文即溢出的意思，在 CSS 中则是指当前元素的内容溢出后应该怎么处理。

    overflow 属性值有：

    • visible：默认值，表示当前元素显示的内容不可截断，这表示它可能会从当前容器元素中溢出，跑到盒模型外面。
    • hidden：此值表示的内容是剪辑，也没有滚动用户界面应提供查看剪辑区域以外的内容。
    • scroll：使用滚动条来显示溢出的内容，使用这一值时，它却不管用户是否需要都使用滚动条机制。
    • auto：自动模式，根据需要使用滚动条。
    • inherit：从父元素继承。
