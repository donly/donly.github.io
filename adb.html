    ADB 找不到设备（Mac OS）

    主要针对国内一些小厂商（杂牌机），Google USB Vendor IDs 列表上没有的设备。

    简单来说，就是将 Vendor ID 添加到 ~/.android/adb_usb.ini，然后重启 adb：

    1. 配置好 adb 环境，开启手机上的 USB Debug 模式。
    2. 将手机通过USB连接到Mac。
    3. 打开系统信息（点击左上的苹果，按住Option会显示），找到如图所示的手机设备，复制厂商ID。
    4. 在终端输入 $ echo 0x1782 >> ~/.android/adb_usb.ini。
    5. 重启 adb：$ ./adb kill-server ; ./adb devices。

    $ ./adb kill-server ; ./adb devices
