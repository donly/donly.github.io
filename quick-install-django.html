    Django 快速安装指南

    Django 是一个高层次的 Python Web 框架，鼓励快速全面的开发和实用的设计。是一个使用 python 语言写的开源 web 开发框架(open source web framework)。

    这是一个简单的快速安装入门教程，如果您需要一个完整的安装指南，涵盖所有安装过程中出现问题的可能性，请参见官方网站：完整安装指南

    安装 Python ¶

    作为一个 Python Web 框架，Django 需要 Python。它适用于任何版本的 Python 2.4 至 2.6，但是它不适用于 3.0。建议您使用 Python 2.5 或更高版本，因为它们包括一个称为轻量级数据库 SQLite。如果您现在使用的是 Linux 或 Mac OS x，则有可能已经安装好了。您可以从这里得到 Python，或者直接进入其FTP服务器下载您想要的版本。

    要验证是否已经安装好 Pyhton，可以在命令窗口中输入"python"，可以看到类似以下的信息说明已经安装成功：
    Python 2.6.5 (r265:79063, Apr 16 2010, 13:57:41) [GCC 4.4.3] on linux2 Type "help", "copyright", "credits" or "license" for more information.

    设置数据库 ¶

    因为我们安装的 Python 是 2.5 以上的版本，里面自带了 SQLite，所以可以跳过这一步。如果需要安装其它数据库，请参考database installation information。

    卸载所有旧的 Django 版本¶

    如果您是从以前的版本升级你的Django的安装，你需要卸载旧版本的Django然后再安装新的版本。

    安装 Django ¶

    安装 Django 的几个步骤：

    • 选择一个为我们操作系统发布的 Django 版本下载，我们以 Django 0.96.5: tarball ， Linux 平台下操作为例。
    • 解压缩并安装： tar xzvf Django-0.96.5.tar.gz cd Django-1.2.1 sudo python setup.py install
    • 验证是否安装成功： >>> import django >>> help(django)

    如果没有报任何出错信息，则说明已经安装成功。

    That is it ¶

    安装成功！赶紧体验 Django 框架带来的快速、简便的 WEB 开发乐趣吧。
